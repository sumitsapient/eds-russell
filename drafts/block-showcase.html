Accordion Block (Default)

What is Edge Delivery Services?

Edge Delivery Services (EDS) is Adobe's modern web delivery platform that serves content from the CDN edge, achieving exceptional performance scores. It uses a document-based authoring model with vanilla JavaScript and CSS.

How does the three-phase loading work?

Eager Phase: Loads only what's needed for LCP — first section, critical CSS.

Lazy Phase: Loads remaining sections, header, footer, non-critical styles.

Delayed Phase: Analytics, martech, and other non-essential scripts load after 3 seconds.

What are Content Fragments?

Content Fragments are structured, headless content items managed in AEM. They have a defined schema (CF Model) and can be queried via GraphQL API. Blocks can fetch and render CF data dynamically, enabling catalog pages, article listings, and more.

How do I create a new block?

Create a folder in blocks/ with the block name, add a .js and .css file with the same name, and export a default decorate(block) function. The EDS framework auto-loads the block when it appears on a page.

Tabs Block (Default)

Overview

Edge Delivery Services combines the authoring power of AEM with blazing-fast delivery. Pages are served from the CDN edge with automatic optimization.

The architecture ensures sub-second load times while maintaining rich authoring capabilities through Universal Editor.

Features

Key features include:

  • Three-phase progressive loading
  • Automatic code splitting per block
  • Built-in Real User Monitoring (RUM)
  • Content Fragment integration
  • Universal Editor for WYSIWYG authoring
Performance

Target metrics:

  • LCP: < 1.5 seconds
  • CLS: < 0.05
  • INP: < 100ms
  • Lighthouse: 100 across all categories
